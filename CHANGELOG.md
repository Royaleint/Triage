# Changelog

All notable changes to Triage will be documented here.
Player-facing language only. For internal/technical details, see `Triage_Dev/CHANGELOG-internal.md`.

## [1.0.0] - Unreleased

### What's New
- **Triage is here.** Enhanced Raid Frames, reforged for Midnight.
- Full support for WoW Retail (12.0+), Classic Era, and Pandaria Classic.
- All original ERF features preserved: 3x3 aura indicator grid, range checking, target markers, frame scaling, profiles.

### Fixed
- Range checking no longer causes errors on Midnight.
- Indicator glow effects restored (broken since patch 11.1.7).
- "Mine Only" filter now correctly shows only your own casts.

### Changed
- Stock auras (Blizzard buff/debuff icons) now visible by default on new installs. You can still hide them in settings.
- Click-casting now works reliably through indicator overlays.
